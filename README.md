#### How to Run

1. Install the appropriate version of chromeDriver from here:
   ```
   https://googlechromelabs.github.io/chrome-for-testing/
   ```

2. Run the following:
   ```
   cd /path/to/folder/
   chmod +x ./intro.bash
   ./intro.bash
   ```
   Alternatively, you can run:
    ```
    python auto-upload-abii-questions.py
    ```
